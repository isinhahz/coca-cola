O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: O git add serve para adicionar alguma alteração feita no seu código. a diferença entre git add <arquivo> e git add . é que o git add <arquivo> adiciona somente uma parte especifica do seu projeto e o git add . seleciona todo seu arquivo por completo.

Por que utilizamos o ponto (.) no comando git add .?
R: Para selecionar todo seu código.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R: O git commit serve para adicionar um comentario, é importante criar uma mensagem descritiva para descrever a atualizaçao feita no seu codigo.

Qual a função do comando git push?
R: O git push serve para enviar como por exemplo um commit para o git hub, esse comando é utilizado para enviar algo para o git hub.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R: É importante criar commits com frequencia para marcar cada etapa feita no seu projeto e caso queira alterar algo no seu projeto essa "divisao" de marcos irá facilitar nas alteraçoes.
